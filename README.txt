Windows:

- Install SWI Prolog: 
  http://www.swi-prolog.org/download/stable

- Go to the folder clingo-5.1.0-win32/third-party/ and run the  Python 2.7.12 installer if Python 2.7.12 is not yet installed.

- Start the Prolog server by double clicking on "prolog-server.pl"

- Open Google Chrome and connect to the Prolog server via: http://localhost:8085/peng/


Linux:

- Install SWI Prolog: 
  http://www.swi-prolog.org/download/stable

- Go to the folder clingo-5.1.0-linux and make sure that you have permissions to execute "clingo.exe".

- Start the Prolog server: double click on "prolog-server.pl".

- Open Google Chrome and connect to the Prolog server via: 
  http://localhost:8085/peng/

- Install SWI Prolog:

- Start the Prolog server: double click on "prolog-server.pl".

- Open Google Chrome and connect to the Prolog server via:
